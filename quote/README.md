# 天翼云等保专区安全产品报价生成器

## 概述

本工具用于根据天翼云等保专区安全产品知识库，自动生成符合要求的Excel报价表。

## 产品分类

### 三级等保基础版（默认最低规格）
- 安全中心（云安全中心-标准版）
- 云安全中心‑日志分析量
- 服务安全卫士‑旗舰版（主机安全-旗舰版）
- Web应用防火墙‑独享版
- 下一代防火墙‑标准版
- 数据库审计
- 日志审计
- 堡垒机
- 漏洞扫描

## 使用方法

运行 `generate_quote.py` 脚本生成报价表：

```bash
py scripts/generate_quote.py [选项]
```

### 选项
- `--full`: 生成全产品报价表（所有产品）
- `--basic`: 生成三级等保基础版报价表（默认）
- `--product <名称>`: 生成指定产品的报价表

## 输出格式

Excel表格包含以下列：
1. 序号
2. 产品规格
3. 详细规格说明
4. 数量（默认1）
5. 标准价-1个月
6. 标准价-1年
7. 折扣价-1年（含包年折扣）
8. 4.5折结算价（包年折扣+4.5折）
9. 5.5折结算价（包年折扣+5.5折）
10. 包年优惠
11. 备注

## 注意事项
- 服务商：LX-2
- 需要绑定弹性IP进行管理，成功开通后请勿解绑该IP
- 需要单独创建一个安全子网（至少保留9个以上可用IP地址）用于部署云等保专区，不能与业务主机所在子网相同，否则可能会存在无法交付的问题
