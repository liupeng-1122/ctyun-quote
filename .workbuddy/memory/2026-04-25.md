## 本地知识库技能部署
- 技能名称: `local-knowledge-base`，已安装到 `~/.workbuddy/skills/local-knowledge-base/`
- 技术方案: TF-IDF + 余弦相似度（纯离线，无需下载外部模型）
- 依赖: scikit-learn, numpy（已安装），可选 pymupdf（PDF支持）
- v1 方案（ChromaDB + sentence-transformers）因国内网络无法下载 HuggingFace 模型而放弃
- 脚本位置: `knowledge-base/scripts/kb_indexer.py`
- 索引数据: `knowledge-base/db/`
- 示例文档: `knowledge-base/inbox/天翼云三级等保产品方案.md`（已成功索引和搜索）
- 搜索命令示例: `py kb_indexer.py search "三级等保物理安全要求"`
- 踩坑: Windows GBK 编码下 print 不能输出 emoji 字符（如 ✅），需用 ASCII 替代
- 踩坑: PDF 文本中含特殊 Unicode 字符（• ● — ' " 等），GBK 编码输出会崩溃，已加入替换逻辑
