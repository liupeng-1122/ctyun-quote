## Excel 输出格式规范（永久有效）

1. **对齐**：自动换行；文本类左对齐+垂直居中。
2. **行高**：根据填充内容自动调整行高（最高不超过20cm）
3. **列宽**：序号 列宽6，其它 根据填充内容自动调整（最宽不超过10cm）。
4. **表头格式**：冻结首行；字体微软雅黑/宋体、12号、加粗、白色；背景色 RGB(59,89,152)；水平+垂直居中；全部框线（外粗内细）。
5. **正文字体**：微软雅黑/宋体、10.5号或11号、常规；金额保留2位小数；日期格式 yyyy-mm-dd。
6. **边框与底纹**：全表完整内外边框；仅表头着色，正文无填充底色；重点数据浅黄底色标注。
7. **其他**：清除多余空格、隐藏行列、无效格式；表格无多余空白行列。

- 定期使用 `kb_indexer.py index --source <文档目录>` 增量更新知识库索引（自动处理新增/修改文档）

- 监控索引文件大小，超过阈值时考虑分库或增大 `max_features`
